== Overview

**VS Code extension providing syntax highlighting, language configuration, and snippets for A2ML (AI-to-Machine Language) files.**

A2ML is the universal manifest format for AI agents, providing structured metadata,
attestation blocks, lifecycle hooks, and gatekeeper configuration for repositories.

== Features

* **TextMate grammar** for `.a2ml` files with full syntax highlighting
* **Language configuration** with bracket matching, comment toggling, and folding
* **7 snippets** for common A2ML patterns (manifests, attestations, SPDX headers, etc.)
* Highlights comments, SPDX headers, key-value pairs, section delimiters, attestation keywords, strings, dates, and URLs

== Installation

Install from the VS Code marketplace:

[source]
----
ext install hyperpolymath.vscode-a2ml
----

Or install from VSIX:

[source,bash]
----
code --install-extension vscode-a2ml-0.1.0.vsix
----

== Highlighted Elements

[cols="1,2"]
|===
|Element |Scope

|Comments (`#`)
|`comment.line`

|SPDX headers
|`keyword.other.spdx`

|Section delimiters (`---`)
|`markup.heading`

|Key names
|`entity.name.tag`

|Attestation keywords
|`keyword.other.attestation`

|Strings
|`string.quoted`

|Dates (ISO 8601)
|`constant.other.date`

|URLs
|`markup.underline.link`

|Booleans
|`constant.language.boolean`
|===

== Snippets

[cols="1,1,2"]
|===
|Prefix |Name |Description

|`spdx`
|SPDX Header
|SPDX license identifier and copyright

|`manifest`
|A2ML Manifest
|Complete AI manifest template

|`attest`
|Attestation Block
|Attestation with trust level and signature

|`gatekeeper`
|Gatekeeper Section
|Gatekeeper protocol configuration

|`---`
|Section Delimiter
|Section delimiter with key

|`kv`
|Key-Value Pair
|Simple key-value pair

|`checkpoint`
|Checkpoint
|Checkpoint entry with status
|===
